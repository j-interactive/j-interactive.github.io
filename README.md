# Joe Griffiths — Personal Profile

A minimal terminal-themed personal profile page.

## Stack

- Vanilla HTML, CSS, JavaScript
- No frameworks or build tools
- Google Fonts: [Share Tech Mono](https://fonts.google.com/specimen/Share+Tech+Mono)

## Features

- Dark/light mode toggle with `localStorage` persistence
- Responsive layout (mobile-friendly)
- Terminal-style UI with green `$` prompt and pill-shaped skill tags

## Structure

```
├── index.html
├── style.css
├── script.js
├── avatar.jpg
└── j-interactive.gif
```

## Usage

Open `index.html` directly in a browser — no build step required.
